🏨 Hotel Management System (Java)

A simple console-based Hotel Management System built using Java.
This project allows users to book hotel rooms, choose meal plans, and generate a detailed bill including check-in and check-out date/time.

🚀 Features

Select area (Ravet, Chinchwad, Hinjewadi, Baner)

Choose hotel from available options

Enter customer details (Name, Address, Aadhar, Age)

Room selection (Luxury, Semi-Luxury, AC, Non-AC)

Meal selection (Combo, Breakfast + Dinner, Non-Veg)

Supports multiple members booking

Enter number of days for stay

Automatic Check-In & Check-Out time generation

Bill calculation with 18% GST

Payment mode: Online (UPI) / Offline

Stores booking details in a file (hotelman.txt)

🧠 Concepts Used

Object-Oriented Programming (Encapsulation)

Java Scanner class for input

File Handling using FileWriter

Java Date & Time API (LocalDateTime, DateTimeFormatter)

Control Statements (if-else, switch)

📂 Project Structure

HotelSystem.java

hotelman.txt (auto-generated after booking)

▶️ How to Run

Clone the repository:

git clone https://github.com/your-username/hotel-management-system.git

Compile the program:

javac HotelSystem.java

Run the program:

java HotelSystem
